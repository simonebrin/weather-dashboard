# Weather-dashboard

## Link to my page: https://simonebrin.github.io/weather-dashboard/

This is a weather dashboard where you can search for the current and 5-day forecast of any city. It will then show you an icon representation of weather conditions, the temperature, the humidity, the wind speed, and the UV index.
